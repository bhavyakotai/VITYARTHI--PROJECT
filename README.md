# Procrastination Predictor using Machine Learning

## Problem Statement
Procrastination is a common issue among students that affects productivity and academic performance. This project aims to predict whether a student is likely to procrastinate based on lifestyle and study habits.

## Solution
A machine learning model is built using Logistic Regression to classify whether a student is likely to procrastinate based on features like sleep hours, screen time, study hours, stress level, and more.

## Dataset
A custom dataset was created with features:
- Sleep hours
- Screen time
- Study hours
- Assignments due
- Social media usage
- Stress level

## Technologies Used
- Python
## How to Run

1. Clone the repository:
   git clone <your-repo-link>

2. Navigate to src folder:
   cd src

3. Install dependencies:
   pip install -r ../requirements.txt
4. Run the program:
   python model.py

## Output
The model predicts whether a student is likely to procrastinate and provides suggestions to improve productivity.
