# Procrastination Predictor

This is a simple machine learning project made using Python. The idea of this project is to check whether a student is likely to procrastinate or not based on their daily routine.

I made this project to understand how machine learning works from scratch without using any special libraries.

## What this project does

This program takes some daily habit inputs like sleep, screen time, study time, etc., and then predicts the chances of procrastination.

It gives the result in percentage form, which shows how likely a person is to delay their work.

## Features used

The model takes the following inputs:

* Sleep hours
* Screen time (in hours)
* Study hours
* Number of assignments
* Social media usage (in hours)
* Stress level (1 to 10)

These values are used to predict the result.

## How it works

First, a small dataset is created inside the program. This dataset contains some example values along with the result (whether a person procrastinated or not).

Then the program trains a simple model using logistic regression. The model is not taken from any library, it is written manually to understand the concept better.

After training, the program asks the user to enter their own details. Based on this input, it calculates a probability and shows the result.

## How to run the project

1. Make sure Python is installed on your system
2. Open the project folder
3. Run the Python file

Example:
python your_file_name.py

4. Enter the values when asked
5. The program will show the result

## Output

The output will be shown like this:

* A percentage value (example: 65%)
* A simple message telling whether you may procrastinate or not
